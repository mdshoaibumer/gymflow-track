# Gym Owner Research

## Target User Profile

| Attribute | Details |
|-----------|---------|
| **Who** | Local gym owner / fitness studio operator |
| **Age** | 28–50 |
| **Tech comfort** | Moderate — uses WhatsApp, UPI, Instagram daily |
| **Gym size** | 20–100 active members |
| **Revenue** | ₹50K–5L/month |
| **Staff** | 1–5 people |
| **Current tools** | Notebook, Excel/Google Sheet, WhatsApp groups |
| **Location** | Tier 1–3 Indian cities |

## Daily Pain Points

### 1. "Who is expiring this week?"
Currently scrolling through a notebook or spreadsheet to find renewals due. Misses renewals = lost revenue.

### 2. "Did this member pay?"
No clear payment history per member. Arguments over "I already paid" are common.

### 3. "Remind members to renew"
Manually typing WhatsApp messages to 10+ people daily. Copy-paste workflow.

### 4. "How much did I earn this month?"
No quick dashboard. Has to manually add up entries.

### 5. "New member joining — where do I store info?"
Either a register book or random WhatsApp message to self. No structured data.

## What They Want

1. **See everything at a glance** — dashboard showing key numbers
2. **Add member in 30 seconds** — name, phone, plan, amount, done
3. **Auto-remind on WhatsApp** — "Your membership expires in 3 days"
4. **Simple payment log** — who paid, when, how much
5. **Works on phone** — they're not always at a desk

## What They DON'T Want

- Complex setup processes
- Feature overload
- Training calls
- Desktop-only software
- English-only (Hindi support = Phase 5)
- Expensive monthly fees

## Key Insight

> The biggest competitor is not other software — it's the **notebook + WhatsApp group** workflow. We must be simpler than that, not just better.

## Onboarding Expectations

- Under 10 minutes to start using
- No credit card required for trial
- Add first member immediately after signup
- See value within first session
- No demo call required
