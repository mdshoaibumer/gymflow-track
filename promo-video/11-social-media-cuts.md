## PLATFORM-SPECIFIC ADAPTATIONS

### Instagram Reels (15s & 30s vertical)
- Captions REQUIRED (85% watch without sound)
- Hook in first 1 second (text overlay)
- Use trending audio snippet underneath if possible
- Add "Save for later 🔖" CTA in last frame
- Hashtags: #gymowner #fitnessbusiness #gymmanagement #saas #gymtech

### TikTok (15s vertical)
- Green screen effect style for screen demos
- Faster pacing — cuts every 1.5-2 seconds
- Trend-aware music (check current trending sounds)
- "Link in bio" CTA
- Text-to-speech option for accessibility

### YouTube Shorts (15s & 60s)
- Subscribe button overlay at end
- Thumbnail optimization for browse
- Loop-friendly ending (last frame ≈ first frame energy)

### LinkedIn (30s landscape)
- More professional tone in text overlays
- Add company logo watermark (top-right, subtle)
- "Learn More" CTA button style
- Focus on ROI messaging: "Save 10+ hours per week"

### Twitter/X (30s landscape)
- Autoplay-optimized: visual hook at 0:00
- Text overlays carry full message (no sound dependency)
- Punchy, quotable one-liner for quote-tweets

### Facebook (30s, 1:1 square)
- Square format for feed dominance
- Subtitles mandatory
- "Learn More" button integration
- Retargeting pixel on landing page
